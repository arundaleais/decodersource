TCP Input

If TCP input is selected, initially AisDecoder will attempt to make a connection to the remote connection as a Client, if this fails it will then try as a Server. These will alternate until successful.

If the remote Server requires authentication by a Username & Password, you must create a Login Command File.

The Login Command File should contain the relevant commands required by the Server to authenticate the user in order for the Server to commence sending data to AisDecoder.

The Login Command File must be named TcpLogin.cmd and must reside in the %APPDATA%\Arundale\Ais Decoder\Files\ directory.

For example file is:-

C:\Documents and Settings\UserName\ApplicationData\Arundale\Ais Decoder\Files\TcpLogin.cmd

Containing:-

User UserName
Pass Password

as required by the Server.

See also Options - Other

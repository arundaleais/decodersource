Summary Window



The Summary window will display the Decoded message summary before input filtering, after input filtering or on being output from the scheduler, dependent on the selection in the Control/Stats Window.

If you click on any line, the Detail Window for the selected sentence will be displayed.

The DAC (Designated Area Code), FI (Function Idendifier), ID (Optional sub-Identifier) are only applicable to ASM's (Application Specific Messages). These can be used to filter input messages.

AisDecoder Help

Updated v145 Feb 2017

Help can be viewed independently of AisDecoder by locating AisDecoder.chm in your Program Files\Arundale\AisDecoder\Help directory and clicking on it.

To print the entire help (approx 80 pages)
1.
 Highlight (click) Ais Decoder Help on Contents tab.
2.
 Right-click and Print
3.
 Select Print the selected heading and all subtopics.

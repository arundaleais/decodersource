Output Options
File Output/UDP Output
 Write data to the Output File/UDP
FTP (File only)
 Send File at Scheduled Interval to FTP Server (eg Website)
NMEA
 Output NMEA data
Time Stamp
 Optionally Time Stamp the NMEA data output
CSV
 Output data as CSV
Head
 Output Header as first line (prefixed with ~ to assist filtering)
Delimiter
 Specify a CSV delimiter (default is comma)
Tagged
 Output Template File with Tags replaced by Tag Values
GIS filtered
 Only output MMSI if vessel's position is known
Scheduled
 Output received sentences at scheduled intervals
Range Filtered
 Apply range filtering to output data

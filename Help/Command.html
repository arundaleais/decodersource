Start Up Command Options

Syntax
"C:\Program Files\Arundale\Ais Decoder\AisDecoder.exe" anoption/anotheroption
Note there is a space between exe" and option
Command Line Options
Option
 Description
/start To automatically start decoding on startup
/nowindow
 To start the decoder without any windows
/ini=full path and filename
 To specify a specific startup initialisation file
/minimise or /minimize
 To Start the decode in the system tray

Example
To start the decoder without any windows (so you can use it as a server)
"C:\Program Files\Arundale\Ais Decoder\AisDecoder.exe" start/ini="C:\Documents and Settings\username\Application Data\Arundale\Ais Decoder\Settings\MyFile.ini"/nowindow

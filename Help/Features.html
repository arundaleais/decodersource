Features

The decoder will decode all the NMEA and AIS content of all 27 AIS message types (!AIVDM or !AIVDO) including many binary messages. A few NMEA GPS sentences are also decoded.

Input: Serial or USB from AIS receiver, UDP or TCP from network, Log File
Input Filter: AIS Message type, DAC, FI, ID, MMSI, $GPxxx (some)
Output: Display, File, FTP, UDP
Output Frequency
 As received or at Scheduled intervals
Output Filter: Lat/Lon or any other decoded field
Output Formats: Text, CSV, HTML, XML, KML, KMZ, NMEA

Details
